# ✊✋✌️ Rock Paper Scissors Console App

> A modernized and refactored C# console implementation of the classic Rock-Paper-Scissors game, focusing on clean code architecture and modularity.

## 📌 Overview
This repository contains a C# console application of the traditional Rock-Paper-Scissors game. Originally built as a foundational programming exercise, the codebase has been completely refactored to demonstrate modern software engineering practices, including method extraction, input validation, and efficient state management.

## 🚀 Features
* **Modular Architecture:** Game logic is cleanly separated from the main application loop for better readability and maintainability.
* **Continuous Gameplay:** Players can enjoy consecutive rounds with a persistent live scoreboard without restarting the application.
* **Robust Input Validation:** Gracefully handles invalid keystrokes and incorrect inputs to prevent application crashes.
* **Memory Efficiency:** Utilizes `ref` parameters for score tracking and a `static readonly` random number generator to ensure optimal performance and true randomness.

## 💻 Installation & Usage

1. Clone the repository.
2. Open the `.sln` file in Visual Studio.
3. Press `F5` to build and run the game.
